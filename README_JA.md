# Telegraph 画像ダウンローダー

Tkinter (customtkinter) ベースのデスクトップツールで、マルチスレッド技術を使用して `telegra.ph` に公開された画像アルバムを一括ダウンロードします。

![](images.png)

## 機能

- **マルチスレッド一括ダウンロード** — 複数リンクの同時ダウンロード対応、スレッド数のカスタマイズ (2〜20)
- **フォーマット変換** — オリジナル、JPG、PNG、WebP、BMP 形式でエクスポート対応
- **ダーク/ライトテーマ** — ワンクリック切り替え、設定の自動記憶
- **ダウンロード履歴** — 最近の100件のダウンロード記録を自動保存
- **リアルタイム進捗** — 各アルバムのダウンロード状況と進捗を表示
- **キーボードショートカット** — `Ctrl+V` リンク貼り付け、`Ctrl+Enter` ダウンロード開始、`Ctrl+S` ログ出力
- **自動リトライ** — ネットワーク不安定時のリトライ機構を内蔵
- **設定の永続化** — 全設定を自動保存し、次回起動時に自動復元

## 使い方

### 動作環境

- Python 3.8+

### 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 起動

```bash
python Telegraph_downloader.py
```

## キーボードショートカット

| ショートカット | 機能 |
|----------------|------|
| `Ctrl+V` | リンクを入力欄に貼り付け |
| `Ctrl+Enter` | ダウンロード開始 |
| `Ctrl+S` | ログをエクスポート |

## ネットワークに関する注意

`telegra.ph` にアクセスするにはプロキシが必要です。

- **macOS**: グローバルプロキシを有効にしてください
- **Windows**: システムのグローバルプロキシが効かない場合があります。`natapp` などの転送ツールをご利用ください
